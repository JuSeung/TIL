# MAC
## 코드 컨벤션 단축키
- command + option + L

## 메서드 선언된 타입으로 변수 생성
- command + option + V

## 세미콜론 자동 생성
- shift + command + enter

## 멀티커서
- shift + command + 8(숫자)

## 최근에 열었단 파일리스트 보기
- command + E

## JAVA 실행
- command + R

## 톰켓 실행
- shift + command + R

## 같은 화면 오른쪽 분할 (사용자 설정)
- shift + option + V

## 분할 포커싱 이동(사용자 설정)
- shift + option + [ (왼쪽)
- shift + option + ] (오른쪽)

## 화면 닫기
- command + W
